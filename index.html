Welcome to Prosys Lab

Our research aims to develop next-generation programming systems for secure and reliable software. We are building real-world software systems including AI-based program analysis systems for detecting deep semantic software bugs, general-purpose program simplification systems for secure and efficient software, scalable program synthesis systems for automatic software generation and repair.
